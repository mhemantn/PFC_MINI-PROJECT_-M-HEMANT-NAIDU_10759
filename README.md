# PFC_MINI-PROJECT_-M-HEMANT-NAIDU_10759


Student Attendance Management System (C)
A simple C program that records, displays, and counts student attendance using a 1D array.
Attendance is marked as 1 for Present and 0 for Absent.
The project demonstrates the use of arrays, loops, and conditional statements.
Tools: C, GCC, VS Code
